# FullStack Blog Application

## Description  
This is a full-stack blog application built with Spring Boot.
Features include:
- User registration and login  
- Creating, editing, and deleting posts  
- Commenting on posts  
- Pagination for better content navigation  

#Technologies
- Spring Boot
- Thymeleaf
- Bootstrap
- MySQL
